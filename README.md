# Clean Slate Computing

> A repository that records and collects **innovative ideas** generated by AI.

`English | `[中文](README.zh-CN.md)

This repo collects meaningful, worth-keeping innovative ideas and knowledge that emerge from conversations with AI. The goal isn't to be "correct" — it's to preserve thinking that steps outside the status quo and reasons from first principles. That is often where new directions begin.

The name **Clean Slate Computing** comes from the first idea collected here: if you could throw away all the historical baggage and design the entire hardware/software stack of the information age from scratch, how would you plan it?
